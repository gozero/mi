# mi

> 模仿小米官网主页，采用html5、css3、ES6、DOM操作、jQuery等技术，利用webpack打包部署网站。

## 演示地址

<https://mynode.applinzi.com>

## 源码

分支为： develop

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm start

# build for production with minification
npm run build
```
